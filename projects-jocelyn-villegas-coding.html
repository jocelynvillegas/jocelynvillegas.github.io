Predicting Gun Types From Documented Gun Violence Incidents in America Between 2013-2018

Developed a machine learning pipeline to classify gun types from a dataset of 200k+ incidents. We focused heavily on feature engineering and data cleaning to standardize inconsistent labels. Achieved optimized results through hyperparameter tuning of SciKit-Learn classifiers. This was an essential hands-on machine learning experience as well as an insightful exploration of various data science tools in python.

Tools Used:

Python, Jupyter Notebook, Pandas, NumPy, SciKit Learn Libraries, GitHub, GitLFS
